# Modern Portfolio

A high-performance, cinematic portfolio interface built with React, Tailwind CSS, and Framer Motion.

## 🚀 Key Features

- **Dynamic Project Synchronization**: Real-time integration with GitHub API.
- **Cinematic Visuals**: Advanced parallax effects and glassmorphism.
- **Performance Optimized**: Lazy loading and optimized assets.
- **Responsive Architecture**: Fully adaptive design for all signal types (devices).

## 🛠️ Tech Stack

- React 19
- Tailwind CSS 4
- Framer Motion
- Vite
