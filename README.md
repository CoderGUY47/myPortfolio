# Modern Portfolio Documentation

Welcome to your premium developer portfolio! This project is built using modern frontend technologies to ensure high performance, responsiveness, and a "WOW" factor for your visitors.

## 🚀 Technologies Used

- **Core**: React.js (Vite)
- **Styling**: Tailwind CSS (Latest), Sass, Vanilla CSS
- **Animations**: Framer Motion (Parallax, Scroll effects, Blobs)
- **Components**: Swiper (Project Slider), Custom React Modals (Popup)
- **Icons**: Font Awesome 6
- **Typography**: Google Fonts (Georama)
- **Assets**: AI-generated high-fidelity images

## 📂 Project Structure

- `/src/components`: Reusable UI components (Hero, About, Skills, etc.)
- `/src/styles`: Global styles and Tailwind configurations.
- `/public`: Static assets including images and the PHP contact handler.
- `/netlify.toml`: Deployment configuration for Netlify.

## 🛠️ Features

- **100% Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Micro-Animations**: Experience smooth transitions and floating elements.
- **Glassmorphism**: Premium look with blurred backgrounds and semi-transparent layers.
- **Grid Skills**: Clean and modern presentation of your technical arsenal.
- **Portfolio PopUp**: YouTube and Image gallery with a sleek modal interface.
- **Contact Form**: Ready for Netlify Forms or a custom PHP backend.

## 🌐 Deployment

This project is configured for **Netlify**.

1. Connect your repository to Netlify.
2. Netlify will automatically detect the `netlify.toml` and run `npm run build`.
3. Your site will be live in minutes!
